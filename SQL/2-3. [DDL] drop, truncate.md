# DDL

- Data Defination Language
- 테이블 생성, 변경, 삭제하는 명령어들
- 앞으로 어떤 데이터를 저장할 것인지에 대한 정의라고 생각하면 됨.

---

# delete datas and table

# truncate

- 데이터만 삭제하는 명령어
- 특정 행만 삭제할 수 없으므로 반드시 모든 행을 통째로 삭제하는 경우에만 사용해야 한다.
- 대용량 삭제할 때 빠른삭제 가능

# drop

- 테이블과 데이터 모두를 삭제하는 명령어

### 둘다 모두 한번 실행하면 돌이킬 수 없으므로 매우 신중하게 사용해야 한다.
