# 시각화 기법
# 1. 단일변수 분석하기 (2가지)

## 1) 연속형데이터: histogram 사용

- 단일변수 + 연속형일 때 사용 <br>
- 분포를 보기 위해서 사용함 : 결측치나 이상값이 있는지 확인 <br>
- 구간의 개수 조절이 가능하다<br>
- 품질관리도구 7개중 하나 (제조업)<br>
- 다양한 패턴 확인<br>

### > 그래프 그리는 방법 (그래프 이해하는 방법)

- 각 구간별 데이터의 개수를 센다.<br>
- 각 구간별 데이터의 개수만큼 위로 쌓아 올린다.
